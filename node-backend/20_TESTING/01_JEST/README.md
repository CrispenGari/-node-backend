### Installations

To install jest we run the following command.

```shell
yarn add -D jest
```

Then you will add the following `script` in the `package.json` file:

```js
{
  "scripts": {
    "test": "jest"
  }
}
```

For typescript types you install them as follows

```shell
yarn add -D @types/jest
```
